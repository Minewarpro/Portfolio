The Maze

Software Used Visual Studio
Languages Used C++

For this project, we had to create a game without using a graphical library. Therefore, we decided to make a maze using ASCII squares.

● Game rules

The maze section

To finish the game, you just need to find the exit marked by the green square. Unfortunately, it's dark, and you have to explore to reveal the path little by little.

You have, to help you, an inventory containing potions and bombs.

The potions allow you to recover health points.

While the bomb allows you to break the squares around you on the horizontal and vertical axes.

The mini-game section

You will encounter blue squares along your path, each representing one of the seven mini-games.

Once the mini-game is finished, you can continue your journey.

If you win, not only will you not lose any health points, but you might also receive a potion or a bomb !

Ylmo

Software Used Phaser
Languages Used JavaScript / html

Play on web

● My first game

This project is my first video game with a beginning, an end, and a lore! We had to create a platformer using the Phaser framework, which is very practical for learning JavaScript coding. However, it also comes with many constraints compared to others. For instance, we could only use rectangular collisions and couldn't rotate them, so no slopes!

I handled everything in this project, from the code to the 2D assets. It allowed me to quickly understand that each role in the video game industry is complex and important. I also realized that the creation process is lengthy, and each task is crucial.

I believe I've grown through this project because I stepped into the other side, transitioning from being a player to becoming a creator.

Promenade

Software Used Unity
Languages Used C#

For this project, we had to think about a serious game centered around the Jardin du Peyrou, a park located in the city of Montpellier, France. We decided to show the player the significant historical stages of this park. So, we came up with the idea of allowing the player to time-travel, landing them at moments marking significant changes in the park.

● Time winding system

Date change

The player can simply increase or decrease the date by pressing the up or down arrows. They can then travel to the chosen date.

● Mini Games

There are three mini-games to complete in order to finish the game :

Hammer | The player must unlock the stone blocking the water flow by clicking at the right timing

The Statue | The player need to quickly tap the screen to fully fill the gauge

Magic Coloring | The player has to color the boxes with the corresponding colors. Each color is associated with a date.
