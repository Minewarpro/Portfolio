The Maze

"Bravez tous les mini-jeux pour trouver la sortie !"
Project Type University
Software Used Visual Studio
Languages Used C++

Gameplay

Details

Sur ce projet nous devions faire un jeu sans bibliothèque graphique. Nous avons donc décidé de faire un labyrinthe fait de carré en ASCII.

Nous étions plusieurs sur ce projet. De mon coté je me suis occupé :

● Des 3C. C'est à dire le déplacement du joueur, comprenant les cases sur lequels on peut aller ou non. Mais aussi le système de point de vie.

● Le système de brouillard se dévoilant lorsque le joueur se déplace.

● Le système d'inventaire et le fonctionnement des items.

● Une partie des Minis-jeux. (Ceux présentés plus bas sont uniquement ceux que j'ai fais)

● Règles du jeu

La partie labyrinthe

Pour finir le jeu il suffit de trouver la sortie marquée par la case verte. Malheureusement il fait noir et vous devez explorer pour devoiler le chemin petit à petit.

Vous avez, pour vous aider, un inventaire contenant des potions et des bombes.

Les potions vous permettent de récupérer des points de vie.

Tandis que la bombe vous permet de casser les cases autour de vous sur les axes horizontal et vertical.

La partie Mini-jeu

Vous rencontrerez sur votre route des cases bleues représentant les minis-jeux. Si vous perdez au mini jeu votre nombre de point de vie diminura alors.

Une fois le mini jeu fini vous pourrez continuer votre route.

Si vous remportez la victoire, en plus de ne pas perdre de point de vie, vous obtiendrez peut être une potion ou une bombe !

● Liste des Minis-jeux

Juste Prix

Les règle de ce mini jeu sont très simples. Vous avez 10 tentatives pour trouver le nombre exact choisi par l'ordinateur.

Suite à chacune de vos réponse il vous sera indiqué si vous etes au dessus ou en dessous . A vous de faire une nouvelle proposition !

Pendu

Celui là est un mini jeu parmi les classique ! L'odinateur choisi un mot qui vous est inconnu et c'est a vous de faire une proposition de lettre.

Si vous tomber juste la lettre est inscrite mais si vous tomber faux 9 fois alors vous perdez la partie !

Morpion

Le Morpion aussi est un grand classique. Alignez 3 croix avant votre adversaire pour remporter la parti !

Les match nul sont considérés comme des victoires.

Remember me

10 secondes, c'est le temps que vous avez pour retenir cette suite de chiffre ! Une fois le temps passé vous devrez réécrir le nombre que vous pensez avoir retenu.

Si vous vous trompez ne serait-ce que d'un seul chiffre, c'est perdu !

Tic Tac

Ce mini jeu est un test de rapidité. Vous aurez 5 secondes pour réécrir le mot affiché.

Soyez rapide mais aussi précis, une seul faute de frappe et c'est perdu !
